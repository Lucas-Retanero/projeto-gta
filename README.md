# GTA V - Rockstar Games Clone

Um projeto de front-end focado em reproduzir a experiência visual e interativa da página oficial de compra do **Grand Theft Auto V**.

Este repositório demonstra a aplicação de fundamentos sólidos de desenvolvimento web, focando em fidelidade ao design original (pixel perfect) e responsividade.

---

## Sobre o Projeto

O objetivo deste projeto foi clonar a interface de compra do site da Rockstar Games. O desafio consistiu em organizar o layout de forma moderna e garantir que as interações de seleção fossem fluidas para o usuário.

**Principais funcionalidades reproduzidas:**
* **Seleção de Plataformas:** Interatividade via JavaScript para alternar entre as versões do jogo.
* **Layout Imersivo:** Uso de backgrounds e tipografia inspirados na identidade visual da Rockstar.
* **Design Responsivo:** Adaptado para proporcionar uma boa experiência em desktops e dispositivos móveis.

---

## Tecnologias Utilizadas

Para manter o projeto leve e focar nos fundamentos, utilizei apenas tecnologias puras:

* **HTML5:** Estruturação semântica e organização do conteúdo.
* **CSS3:** Estilização avançada, utilizando Flexbox e Grid para o layout.
* **JavaScript (Vanilla):** Lógica de interação com o DOM para os botões de seleção e estados da página.
